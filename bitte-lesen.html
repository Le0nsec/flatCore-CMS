flatCore - Content Management System

Installation

Die Installation dauert nur wenige Minuten. Bevor Du mit dem Upload der Daten auf den Webserver beginnen kannst, musst Du in der Datei config.php ein paar Werte anpassen.

1. mod_rewrite

flatCore kann sowohl mit als auch ohne mod_rewrite betrieben werden. Wird mod_rewrite aktiviert, erzeugt flatCore schönere URLs für Dein Projekt.

Beispiel mit mod_rewrite: http://example.com/news/

$fc_mod_rewrite = "permalink";

oder

Beispiel ohne mod_rewrite: http://example.com/index.php?p=2

$fc_mod_rewrite = "off";

Damit mod_rewrite funktioniert, muss die Datei _htaccess in .htaccess umbenannt werden. Dateinamen die mit einem Punkt beginnen sind bei den meißten Systemen für das System selbst vorbehalten. Darum ist es am einfachsten Du benennst die Datei erst um, wenn Du sie auf den Server kopiert hast.

2. E-Mail Absender

Für Benachrichtigungen, Aktivierungslinks etc. versendet flatCore E-Mails. Damit die Empfänger sofort erkennen, woher die E-Mail kommt, solltest Du aussagekräftige Angaben wählen.

$fc_mailer_adr = "info@example.com";
$fc_mailer_name = "Example Administrator";

Jetzt kannst Du die Dateien und Verzeichnisse auf Deinen Server kopieren und das Installationsscript starten.

3. Installieren

  1. Alle Dateien auf den Server kopieren (kann auch in ein Unterverzeichnis sein).
  2. Im Webbrowser das Verzeichnis /install/ aufrufen und den Anweisungen folgen.
    • Die gewünschten Daten für einen Administrator eingeben.
    • Auf "Installation starten" klicken.
  3. Das Verzeichnis /install/ löschen oder umbenennen.
    • Solange das Verzeichnis /install/ existiert, kann das Frontend nicht aufgerufen werden.

War die Installation erfolgreich, führt der nächste Weg in das ACP.
Im Browser das Verzeinis /acp/ aufrufen und mit den Zugangsdaten des Administrators anmelden.
Beispiel: http://www.example.com/acp/
