blucent

The default flatCore Theme

Optionen

Cookies

Wenn du einen Hinweis zu Verwendung von Cookies auf deiner Website anzeigen möchtest, musst du nur eine Textvorlage erstellen und diese privacy_policy benennen. Der Hinweis wird dann automatisch erzeugt.

Teaser

Das Template layout_portal_teaser.tpl zeigt als großen Banner das Bild blucent/images/teaser.png. Um dieses Bild zu ersetzen, musst du ein Bild mit dem Namen teaser_background_image.png hochladen.

Als Titel/Untertitel des Teasers wird der Seitentitel und die Seitenbeschreibung (Meta-Tags) genommen.

Cookies

If you want to show an alert about using cookies on your site, you just have to create a text template and name it privacy_policy. The alert will be automatically generated.

Teaser

The Template layout_portal_teaser.tpl uses the image blucent/images/teaser.png. You can replace this Teaser-Image by uploading a new image named teaser_background_image.png.

The Caption is generated from the page title and description (Meta-Tags).
