Dieser Ordner wird u.a. für statische HTML Seiten verwendet.
Diese Seiten können über das Content Management System nicht bearbeitet werden. Das wird sich auch nicht ändern, denn es ist absicht!

Um eine HTML Seite aus diesem Ordner mit den Seiten aus dem Content Management System zu verknüpfen, kann man entweder schlicht einen Link setzen oder die Datei per include() in eine Seite integrieren.
